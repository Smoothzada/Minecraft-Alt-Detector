
# Minecraft Alt Detector

Written in **Python**, this is a tool to check suspicious Minecraft accounts. While primarily designed for use in **Prime RBW**, anyone is welcome to use it.

# Privacy & Security

For more details, the source code is available above.
- No data, logs, or results are ever sent to external servers.
- The tool only collects **usernames/nicknames**.
- **No internet connection required** (the tool works entirely offline).

# Supported Clients

Currently, the Alt Detector works with:
- Forge
- Badlion
- Feather
- Lunar (including Celestial Launcher)
- TLauncher

# Usage

### Compiled Version
You can download the compiled version [here](https://github.com/Smoothzada/Minecraft-Alt-Detector/releases/tag/Compiled).

### From Source
If you want to clone the repository, use the following command:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Smoothzada/Minecraft-Alt-Detector.git
   ```

2. **Navigate to the folder:**
   ```bash
   cd Minecraft-Alt-Detector
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the tool**
    ```python
    py main/main.py
    ```
